# Company Profile Website

## Overview
This project is a simple yet elegant company profile website that showcases the company, its products, and provides contact information along with a Google Maps location. The website is designed to be visually appealing and user-friendly.

## Project Structure
The project consists of the following files and directories:

- `index.html`: The main HTML document for the website.
- `css/styles.css`: Contains styles for the website, defining layout, colors, and fonts.
- `js/main.js`: JavaScript code for interactive features and loading product data.
- `data/products.json`: JSON file containing product information such as names, descriptions, prices, and images.
- `.gitignore`: Specifies files and directories to be ignored by Git.
- `README.md`: Documentation for the project.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to view the website.
3. For development, you can edit the CSS and JavaScript files as needed.

## Features
- Homepage showcasing the company.
- Section explaining the products sold.
- Google Maps integration for location display.
- Contact information section.
